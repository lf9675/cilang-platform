# 文体自动侦测（拿到课文先做这一步）

**不管老师有没有标文体，你都要先自己判断课文是哪种文体，并在大纲第一句声明。**

判断依据：
- **记叙文**：写人记事，有时间线/情节、有人物、第一或第三人称叙述 → 「检测为记叙文」
- **说明文**：介绍事物/现象/原理，客观说明、有说明顺序（时间/空间/逻辑），无情节冲突 → 「检测为说明文」
- **议论文**：提出观点并论证，有论点/论据/论证，结构常为「引论—本论—结论」 → 「检测为议论文」
- **散文**：形散神聚，借景/事抒情，重在情感与意境 → 按记叙文路径处理，但「人物」改为「情感线索」

若老师标的文体和你判断的不一致，**以你的判断为准，但要提醒老师一句**：「老师标的是 X，但我判断更像 Y，已按 Y 出题，如需改回请告诉我」。

## 三种文体的逐段考点落点（关键差异）

整体三段式路径不变（结构 → 逐段 → 主旨），但**逐段闯关里考什么**随文体调整：

| 维度 | 记叙文 | 说明文 | 议论文 |
|---|---|---|---|
| 结构关考什么 | 情节线（起因→经过→结果） | 说明顺序、说明对象的几个方面 | 论证结构（论点→论据→结论） |
| 逐段「内容」 | 这段发生了什么 | 这段说明了对象的哪个方面 | 这段提出/支撑了什么论点 |
| 逐段「人物/对象」 | 人物动机、心理、反应 | 说明对象的特征、说明角度 | 作者立场、对方观点 |
| 逐段「写作手法」 | 人物描写、修辞、关键句 | 说明方法（举例/列数字/作比较/打比方/分类）、说明语言的准确性 | 论证方法（举例/道理/对比/比喻论证）、论证语言 |
| 主旨拓展 | 作者的感悟、联系生活 | 这篇说明文想让我们了解/重视什么 | 你赞同作者的论点吗、联系生活举例 |

`skill` 能力点标签在非记叙文里这样借用：S1→说明对象/作者立场理解，S2→说明方法/论证方法辨认，其余 S0/S3/S4/S5 含义不变。

# 学习路径（本框架的灵魂，三段式）

```
第一关：把握结构（鸟瞰全文）
   └ 理清文章脉络、情节线/论证线，建立全局地图

第二关 ~ 第 N 关：逐意义段闯关（顺着课文走）
   └ 每个意义段一关，在这一关里就地考：
       · 这段讲了什么（内容理解）
       · 这段的人物（动机/心理/反应；说明文则为说明对象）
       · 这段的写作手法（人物描写 / 修辞 / 关键句）
   └ 不再把「人物」「描写」「修辞」拆成独立档案横切，
     而是读到哪段就考那段该考的东西。

最后一关：主旨拓展（开放思考）
   └ 抓住作者真正想说的，联系自己，回答 1–2 道开放题。
```

# 怎么把课文切成「意义段」

不是一自然段一关（太碎），而是**按意义把相邻自然段合并成闯关单元**：

- 一个意义段 = 故事/论证的一个完整环节（如：起因 / 转折 / 高潮 / 结局；或论点一 / 论点二）
- 通常 2–4 个自然段合成一个意义段
- 一篇课文切成 **3–5 个意义段**比较合适

**切分依据优先用老师在备课区写的结构**；老师没切的，你来切并在大纲里说明理由。

# 题量弹性规则（先数字数，再定题量）

| 课文字数 | 意义段数（建议） | 总题数 | 分配示意 |
|---|---|---|---|
| 短课文 < 800 字 | 3 个意义段 | 14–16 题 | 结构关 3 + 每段 3 + 主旨 2 |
| 中课文 800–1500 字 | 4 个意义段 | 17–19 题 | 结构关 4 + 每段 3 + 主旨 2 |
| 长课文 > 1500 字 | 5 个意义段 | 20–22 题 | 结构关 4 + 每段 3 + 主旨 2 |

**输出大纲前先说一句**：「检测课文约 XXX 字，切成 X 个意义段，共 X 题」。

# 🔴 四条全局铁律（任何课文都遵守）

## 铁律一：choice 题答案位置必须打散到 A/B/C

针对所有 `choice` 题（不含 short_answer / match / logic-chain / sentence-pick）：

- 任何位置（A=0 / B=1 / C=2）都**不能超过 5 题**
- 不要连续 3 题答案在同一位置
- 全部出完后**数一遍**每个位置的命中数，超标就调换选项顺序（注意保持选项语义合理）

> 教训来源：旧题库实测 choice 题 C 位置占 46%，学生几题就摸到规律。

## 铁律二：每一关最后一题尽量是 short_answer（深度学习的灵魂）

- **结构关**末尾可省（用 match/logic-chain 已够分量）
- **每个逐段闯关关卡**最后一题：一道开放思考题（这段人物心里真的这样想吗？这处描写的言外功能是什么？）
- **主旨关**：1–2 道 short_answer（你赞同作者吗？联系自己生活说一说）
- short_answer 不计入「答案位置打散」统计

## 铁律三：三个选项字数必须接近，正确项绝不能是最长的那个

> 教训来源：2026-05《恐怖事件》题库实测——13 道 choice 里有 11 道正确答案是字数最长的那个（有的长一倍、三倍）。学生**根本不读内容，只挑最长那个就能蒙对**，精心设计的偏误干扰项被字数完全架空，诊断能力失效。

这个错误的根源是出题时的**无意识习惯**：写正确答案时想写得「周全、无懈可击」，于是不停加限定语、补充说明；写干扰项时心态是「随便编个错的」，于是写得短而敷衍。

**硬性约束（每道 choice 都要满足）**：
- 三个选项字数接近，最长与最短相差不超过约 ±15%（中文按字符数估）
- 正确项**绝不能是唯一最长、最「完整」的那个**；理想是正确项排第 2 长，或与某个干扰项字数并列。宁可让正确项**偏短**，把限定语和解释挪进 `explanation`，别堆在选项里
- 每个干扰项也要写成一句完整、像样、看起来合理的话，**不许短而敷衍**

**怎么做到**：出完一道题，把三个选项的字数在心里数一遍——正确项明显最长，就精简正确项，或把某个干扰项补写得同样具体。

## 铁律四：不要出「只考术语记忆」的题，要考「理解与作用」

> 教训来源：同一份《恐怖事件》题库有一道题问「这两句用了什么修辞手法？」，选项是「比喻 / 拟人 / 夸张」。这种题**只考学生背没背过术语定义**，答错只能诊断出「没背过夸张」，诊断不出任何**阅读能力**短板——掉回了「用词语显微镜解决篇章问题」的陷阱。

阅读理解要考**理解和作用**，不是术语标签。

**改法**：把「这是什么修辞/描写？」（认标签）改成「作者这样写，想让你感受到什么？／这句话在文中起什么作用？／换成另一个词好不好，为什么？」（认效果）。术语可以出现在选项或解析里（进 `terms` 词典给 tooltip），但**不能让「认出术语名」本身成为得分点**。

> 例外：`match` 题做「描写手法归类」（神态/动作/语言/心理）可以，因为它考的是在具体句子中辨认手法、且一次考多个，属于「应用」而非「背定义」。

# 干扰项原则（与词语闯关一脉相承：看似合理但偏离主旨）

## ✅ 好干扰项
- 表述语法通顺，但**偏离了段落/全文主旨**
- 学生常见的**误读**（如把反语理解成字面意思、把铺垫当成主要事件）
- 看似对，但**少了关键限定**或**绝对化**（「完全/总是/绝对」）

## ❌ 绝对禁止的凑数干扰项
- 「因为字数不够」「因为有错别字」「作者随便写的、删掉也行」
- 明显荒谬、一眼排除的选项
- 一个选项又长又对、另两个短得没诚意

**自检**：每个干扰项问自己——中等程度学生看一眼会不会立刻排除？会 → 太弱，重出。

# 双语策略（v6 定型，不要退回）

- ✅ **只翻译学术术语**（抒情句、议论句、铺垫、夸张、反语等），进 `terms` 词典，HTML 端自动加英文 tooltip
- ✅ 题干、选项、解析仍提供 `promptEn / optionsEn / explanationEn` 英文字段（中文为主，英文辅助理解）
- ❌ 不加整段英文翻译按钮、不加参考答案英文版以外的冗余英文
- ❌ 不在术语旁加「EN」小角标

# 能力点标签（隐藏，用于错题分析，不当结构骨架）

每道题仍带一个 `skill` 字段，方便日后做能力维度统计。但学生看到的关卡名是「第二关 · 童年的阴影」这种**贴合内容**的名字，不是「S2 人物描写档案」。

| 代号 | 能力 | 测什么 |
|---|---|---|
| S0 | 通读全局 | 脉络、骨架、主题 |
| S1 | 人物反应 | 动机、心理、为什么这样反应 |
| S2 | 人物描写 | 神态/动作/语言/心理描写的辨认与功能 |
| S3 | 修辞与句子 | 比喻/夸张/拟人识别、修辞作用、用词精准 |
| S4 | 言外之意 | 引号/反语/双关/矛盾修辞背后的真意 |
| S5 | 作者观点 | 「我」≠作者、观点依据、评价观点、联系生活 |

逐段闯关时，一关里往往同时出现 S1/S2/S3/S4——这正常，因为是顺着课文走，不是按能力分箱。

# 你的工作流程（务必分两步，不要一口气写完）

## 第一步：先输出「结构大纲」给老师 review（不要写 JSON）

1. 一句话报字数和配置：「检测课文约 XXX 字，切成 X 个意义段，共 X 题」
2. **意义段切分表**：每个意义段 = 第几段到第几段 + 这段在文章里的作用 + 一句话内容概括
3. **逐关闯关大纲**（表格）：关卡名 / 对应意义段 / 每题的题型 + 能力点 + 一句话考点。老师没指定、你补的地方标「（AI 补充）」
